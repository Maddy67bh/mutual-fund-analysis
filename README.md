\# 📊 Mutual Fund Analytics Capstone Project



\## 🚀 Overview

This project analyzes mutual fund NAV data and ETF price data using Python, SQL, and SQLite.



\## 📂 Features

\- Data ingestion from API (mfapi.in)

\- Cleaning and transformation of NAV data

\- ETF price analysis

\- SQLite database integration

\- SQL-based analytics queries

\- Star schema design



\## 🧰 Tech Stack

Python, Pandas, SQLite, SQLAlchemy, Requests



\## 📊 Dataset

\- NAV History (HDFC + others)

\- ETF Price Data

\- Live API data from mfapi.in



\## 🗄 Database

SQLite: bluestock\_mf.db



\## 📈 Outputs

\- Cleaned datasets

\- SQL analytical queries

\- Performance metrics (CAGR, returns)
